BeerFinderNZ

This application allows you to obtain a list of stores with the best beer prices in New Zealand, filtering by island, region and city and brand.

Using a PostgreSQL database that contains information about the pages of more than 190 stores, from 4 different franchises in 91 cities in 15 regions, this application obtains data through asynchronously web scrapping and returns a list ordered by lowest price.

To know more about this project please read the README in the Github repository.
