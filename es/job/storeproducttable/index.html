StoreProductTable

Este proyecto fue mi segundo trabajo por encargo, es una aplicación para tablets android que se utiliza para crear una tabla de prendas, llevar un control sobre los precios y permite calcular y aplicar un aumento a los precios mediante porcentajes, de forma general o especifica utilizando una serie de filtros.

Esta aplicación almacena las listas de prendas en archivos locales y permite la creación de copias de seguridad entre otras funcionalidades.
