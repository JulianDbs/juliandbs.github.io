    Inicio

ShareYourFiles

Este proyecto de código abierto nació de la necesidad de entregar software privativo a un cliente sin almacenarlo en un servicio de almacenamiento online del cual el cliente no es propietario, como Google Drive, Mega, etc.

Dado que los servidores FTP actualmente se consideran inseguros, la idea fue crear un servicio que pudiera ser desplegado en un servidor linux de forma razonablemente sencilla y permitiera al desarrollador, mediante una interface web, crear una cuenta para el cliente y darle acceso temporal a carpetas y archivos.

Para mas información lea el README del repositorio de Github.
